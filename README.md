# 数据挖掘项目合集
作者：江晓珊
方向：数据挖掘 / 机器学习 / 统计建模 / 多源数据处理

## 项目1：时序碳排放预测
- 数据规模：10万+条时序数据
- 使用工具：Python、Pandas、Scikit-learn
- 模型：随机森林、GBDT、ARIMA
- 评估指标：MAE、RMSE
- 任务：数据清洗、时序特征工程、模型对比与预测分析

## 项目2：农业病虫害图像分类
- 任务：多模态数据处理、图像分类与智能问答
- 效果：分类准确率 85%
- 赛事：第七届全球校园人工智能算法精英大赛（全国总决赛二等奖）

## 项目3：统计建模竞赛数据分析
- 任务：多表合并、因素分析、风险因素建模
- 使用工具：Python、Jupyter Notebook
- 产出：完整数据分析流程、建模报告与项目方案
